HOW TO USE THIS:

INSTALL AND ENABLE THE PLG_IP_SAS_ADMIN.ZIP PLUGIN.

CONFIGURE THE OPTIONS IN THE PLUGIN SETTINGS.

TO SET THE AVAILABILITY CALENDAR FOR A LISTING, VISIT THE LISTING AS THE LISTING AGENT, SUPER AGENT OR ADMIN (ANYONE WITH ACCESS TO EITHER THE ADMIN OF IPROPERTY OR TO THE EDIT PAGE FOR THE LISTING).

A NEW TAB WILL BE SEEN INSIDE THE PROPERTY EDIT FORM, WITH "AVAILABILITY" AS THE DEFAULT NAME (THIS CAN BE CHANGED IN THE PLUGIN PARAMETERS).

CLICK ON A CALENDAR DATE TO CHANGE ITS STATUS. CLICK MULTIPLE TIMES TO SWITCH BETWEEN VALID STATUSES.
